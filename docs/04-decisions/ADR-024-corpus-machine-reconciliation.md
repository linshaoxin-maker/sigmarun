# ADR-024：语料库保持宪法权威 + 机器对账

- 状态：已采纳（2026-07-15，D24）
- 背景：D20 之后三波交付只进 CHANGELOG，语料库/事件目录/决策台账集体失鲜——漂移根因是「对账靠人」。
- 决定：语料库仍是唯一权威；恢复「每特性带回写」；四张机器对账常驻 CI——①命令面 ↔ docs/17 §1（COMMAND_SURFACE 双向包含 + 每条真实可派发）；②exit 码 ↔ docs/17 §2.2；③发出事件集 ↔ docs/18 §2 活动目录（双向相等）；④依赖矩阵/单一事务骨架/单一 EVENT_STATUS（architecture.test.ts）。
- 后果：下一次「特性只进 CHANGELOG」直接把 CI 打红。
